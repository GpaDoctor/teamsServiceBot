# teamsServiceBot
# Company policy
Turn on wifi\

# Virtual environment
Inside project folder
```
python -m venv venv
```
# Activate
Windows:
```
.\venv\Scripts\Activate.ps1
```
Linux
```
source venv/bin/activate
```

# Run installation
```
pip install
```
Create .gitignore
```
ni .gitignore
```

Create requirements.txt
```
pip freeze > requirements.txt
```
